User Feedback System

User Feedback system will provide a platform to enable ecommerce businesses get feedback on the services and products they are offering. It will provide a comprehensive set of tools to help in all phases of a feedback loop, starting from gathering of the feedback, performing analytics on the provided feedback to informing users once the suggestion has been implemented and available for consumption.
